Если текущий символ точка (.) то проверяем какой следующий символ:
    1. Звездочка.  В структуру во флаг dot записываем аргумент из va_args
    2. Цифра. в значение флага dot записывает текущую цифру в формате int (переводим из str по аналогии с atoi)

Если текущий символ минус (-), то переключает флаги в структуре - флаг minus = 1, а флаг zero = 0, т.к. в случае есть есть флаг минус то флаг 0 игнорируется.

Если текущий символ звездочка (*), то мы считаем что это флаг ширины, переключает флаг star = 1, и во флаг width - считываем значения из va_arg.
Проверяем значение полученное из va_arg - возможны два варианта:
    1. Отрицательное - минус считаем как флаг минус (-) и переключаем флаг minus в значение -1
    2. Любые другие значение. Ничего не делаем.

Если текущий символ цифра - проверяем флаг star:
    1. Установлен, т.е. star = 1. Тогда записываем во флаг width 0, а потом туда же записываем 0, только переведенный из ascii кода.
    2. Не установлен, т.е. не равен 1. Тогда записываем во флаг width значение ширины, которое получается переводом строчного значения в int(как в atoi)
